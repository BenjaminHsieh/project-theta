## Data Analysis

The Makefile contains the following commands: `clean`, `test`, `verbose` and`coverage`. 

- `make test`: Runs function tests on functions directory in utils that are associated with analyzing the data. 
- `make verbose`: Verbose version of `make test`.
- `make clean`: Cleans for temporary files generate by script runs or test runs.
- `make coverage`: Generates a coverage report of functions used in analysis.
